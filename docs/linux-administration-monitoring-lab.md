# Linux Administration and Monitoring Lab

## Overview

The Linux Administration and Monitoring Lab documents hands-on work with Linux system baselining, command-line administration, monitoring, troubleshooting, and technical documentation.

## Purpose

This lab demonstrates practical Linux skills that support IT support, service operations, systems administration, and technical troubleshooting roles.

## Core Activities

- System information review
- Package and service review
- File and directory management
- Log review
- Basic monitoring
- Troubleshooting documentation
- Baseline documentation

