# Network Discovery and Analysis Lab

## Overview

The Network Discovery and Analysis Lab documents hands-on work with network visibility, device inventory, service discovery, and network documentation.

## Purpose

This lab demonstrates practical networking skills that support IT support, help desk, technical support, systems administration, and cybersecurity learning.

## Core Activities

- Device inventory
- Network mapping
- IP address review
- Service discovery
- Port documentation
- Basic network observations
- Diagram planning

## Workflow

```text
Network Scope Defined
        ↓
Devices Identified
        ↓
Services Reviewed
        ↓
Findings Documented
        ↓
Diagram Created
        ↓
Recommendations Added
```
